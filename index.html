Heart disease Using Machine Learning

Predicting cardiac disease can play an important role in prevention, treatment, and providing critical support to healthcare professionals in their efforts to combat this condition. This work focuses on developing a predictive model for heart disease using recent advances in machine learning. Multiple datasets from trustworthy sources, including the UCI repository and Kaggle, contain roughly 13 features relevant to patients' medical histories, encompassing both individuals diagnosed with and without heart disease. For model development, machine learning methods such as Random Forest, Support Vector Machine (SVM), K-Nearest Neighbors (KNN), Naive Bayes, Decision Tree, and XGBoost are employed.
